In this assignment, we will implement a data structure to represent a rational number p/q, and some basic operations on them.

Rational Number Library

Create a library to support the following operations on rational numbers:
  1. rational rational_parse( char* str ): Read a string "str" that contains the rational number in a "p/q" format into your data structure. Make sure you reduce this to the simplest form.
  2. void rational_print( rational x ): Print the rational number x in the format "p/q" to the terminal (using printf)
  3. rational rational_from_int( int p ): Convert an integer to a rational number (p/1).
  4. double rational_to_double( rational p ): Convert a rational number p to a real number (of double type).
  5. rational rational_add( rational p, rational q ): Add two rational numbers p, q. Make sure you reduce the resultant sum.
  6. rational rational_sub( rational p, rational q ): Subtract two rational numbers p, q. Make sure you reduce the resultant sum.
  7. rational rational_mul( rational p, rational q ): Multiply two rational numbers p, q. Make sure you reduce the resultant.
  8. rational rational_div( rational p, rational q ): Divide two rational numbers p, q. Make sure you reduce the resultant.
  9. rational rational_reciprocate( rational p ): Compute "1/p".

Continued Fractions

In mathematics, an infinite continued fraction is an infinite expression obtained through an iterative process of representing a number as the sum of its integer part and the reciprocal of another number, then writing this other number as the sum of its integer part and another reciprocal, and so on. In this exercise, we will use the rational number library you just wrote to compute some continued fractions. You may refer Wikipedia for the form of the continued fractions.
  1. A finite continued fraction can be represented as a list a0, a1, a2, a3, and is computed as a0 + 1/( a1 + 1/( a2 + 1/a3 ) ). Write a program that will allow the user to enter 4 integers for a0, a1, a2, a3, respectively, and will print the double value of the appropriate continued fraction.
    Input:200 0 10 8
    Output:210.125000
  2. π can be expressed as an infinite continued fraction, with ai = [3,7,15,1,292,1,1,1,2,1,3,1,14,2,1,1,2,2,2,2,1,84,...]. Let's study how the accuracy of the continued fraction improves as you consider longer sequences. Compute the continued fraction from the first 'k' values of ai (given as user input), and print the relative error with respect to π from math.h, in scientific notation (i.e. use %e).
    Input:4
    Output:-8.491368e-08
