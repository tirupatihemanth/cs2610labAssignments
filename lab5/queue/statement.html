Queues:

The queue is another data structure that you will use very frequently. Queues are very popular in a lot of applications and in resource management systems.

In its barest form, a queue can also be implemented as a singly-linked list which allows you to add elements only to the end of the list and elements can be removed only from the front of the list.

Implement a queue with at least the following operations:

  • Queue* queue_new(): Creates a new, empty queue.
  • Queue* queue_new(): Creates a new, empty queue.
  • void queue_delete( Queue* q ): Deletes the queue, freeing all memory.
  • Queue* enqueue( Queue* q, int val ): Inserts val to the rear of the queue. Should return q (Note: By returning the same input, you can chain together many functions like queue_dequeue( queue_enqueue( q ) ), etc. ).
  • int queue_front( Queue* q, int* error ): Returns the element currently at the front of the queue. If the queue is empty, error should be set to 1, else 0. (Note: error is an argument passed by reference - give it the address of a locally defined integer.)
  • int queue_rear( Queue* q, int* error ): Returns the element currently at the rear-end of the queue. If the queue is empty, error should be set to 1, else 0. (Note: error is an argument passed by reference - give it the address of a locally defined integer.)
  • Queue* dequeue( Queue* q ): Deletes the value at the front of the queue. Remember to free memory.
  • int queue_size( Queue* q ): Return the number of elements in the queue.
  • void queue_print( Queue* q ): Iterates over the entire queue, printing its elements one by one - separated by spaces.
