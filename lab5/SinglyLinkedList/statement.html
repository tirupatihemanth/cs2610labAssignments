Linked Lists

Implement a linked list with at least the following operations:

  • LList* llist_new() Create an empty list (head shall be NULL)
  • int llist_size( LList* list ) Traverse the linked list and return its size
  • void llist_print( LList* list ) Traverse the linked list and print each element
  • LList* llist_append( LList* lst, int data ) Add a new element at the end of the list
  • LList* llist_prepend( LList* lst, int data ) Add a new element at the beginning of the list
  • LList* llist_insert( LList* lst, int idx, int data ) Add a new element at the @idx index
  • LList* llist_remove_last( LList* lst ) Remove an element from the end of the list
  • LList* llist_remove_first( LList* lst ) Remove an element from the beginning of the list
  • LList* llist_remove( LList* lst, int idx ) Remove an element from an arbitrary position in the list

List Command Shell

A basic command interface has been written for you in main.c, which allows you to enter commands like append 1, etc. to append 1 to the list. As you implement the above functions, uncomment the corresponding function calls to them (these have been marked with a NOTE), and test them out. Once you are done, remove the debug print statements (also marked with a NOTE).

The following commands are supported by the basic shell,

  • quit: Leave the command shell. Pressing "Ctrl+D" also quits the command shell.
  • count: Print the number of elements in the linked list
  • append x: Appends the integer 'x'.
  • prepend x: Prepends the integer 'x'.
  • insert x y: Prepends the integer 'y' at position 'x' in the list. Position 0 is the very beginning of the list, and insert 0 x is the same as prepend x.
  • remove-last: Remove the last element of the list.
  • remove-first: Remove the first element of the list.
  • remove x: Remove the 'x'-th element of the list.
  • print x: Print the 'x'-th element of the list.

Polynomial Differentiation and Integration

Use the Linked List ADT implementation to represent a polynomial as a linked list, where each term in the polynomial is represented by a node. The data field must encode both the coefficient as well as the exponent. The header node will have the term corresponding to the highest polynomial exponent (i.e degree) and the next pointer should point to the node corresponding to the term whose exponent is just lesser than the current node.

Constraints

Let d be the degree of the polynomial and a be any coefficient. The range of values for d, a are:

0 ≤ d ≤ 999
-999 ≤ a ≤ 999

Input/Output format

The polynomial [axn+bx(n-1)+cx(n-2)+...+z] must be represented in the following format in input/output files:
a n
b n-1
c n-2
.
.
z 0

 Modify the Makefile to create an executable named polynomial. In polynomial.c, implement the following functions:

  • polynomial polynomial_construct(): Read the input file given in the above format, construct a linked list
  • polynomial differentiate(polynomial a): Perform differentiation of the input polynomial.
  • polynomial integrate(polynomial a): Perform integration of the input polynomial.
  • polynomial add_polynomial(polynomial a, polynomial b): Add polynomial a and b.
  • polynomial subtract_polynomial(polynomial a, polynomial b): Subtract polynomial b from a.
  • void print_polynomial(polynomial a): Print the polynomial in the above format.
In main2.c, read a polynomial and calculate the differential, integral, sum of the differential and integral, difference between the differential and integral. Display the four resulting polynomials. Leave one line space between each result.
